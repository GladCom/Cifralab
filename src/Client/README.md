## Команды
Для управления проектом используется Makefile, все команды описаны в нем. 
Применение команд: 
1) убедитесь, что находитесь в директории, в которой расположен Makefile
2) Введите "make <команда из списка>"

Подробнее про [Makefile](https://guides.hexlet.io/ru/makefile-as-task-runner/)
